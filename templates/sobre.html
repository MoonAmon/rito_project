Sobre o projeto:

O objetivo principal do site é mostrar alguns dados relacionados as últimas partidas jogadas pelos players num jogo chamado League Of Legends.

Os dados foram fornecidos através de uma api fornecida pela riot games (desenvolvedora do jogo). Essas informações foram implementadas através da programação orientada à objetos, utilizando a linguagem python.
